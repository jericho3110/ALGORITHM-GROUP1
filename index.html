1. HTML Text Elements & Formatting

HTML provides various elements to format text effectively:

Key Concepts

  • Text Elements: Use headings (<h1> to <h6>), paragraphs (<p>), and lists (<ul>, <ol>, <li>).
  • Inline Formatting: <strong> (bold), <em> (italic), <u> (underline), <mark> (highlight), <del> (strikethrough), <code> (inline code), <br> (line break), and <hr> (horizontal rule).
  • Semantic Formatting: Use <strong> instead of <b> and <em> instead of <i> for better meaning.
  • Whitespace Handling: Use <pre> to preserve spaces and line breaks.

Example Code:

<h1> Welcome to My Webpage </h1> <p> This is a <strong> bold statement </strong> and an <em> italicized note </em>.</p> <ul> <li>Item 1</li> <li>Item 2 <mark> highlighted </mark></li> </ul>

2. Attributes & Linking HTML Files

Attributes provide extra information to elements.

Key Concepts

  • Attributes: Elements can have attributes like id, class, href, and target.
  • Linking:
    • External Links: <a href="https://example.com">Visit Example</a>
    • Internal Links: <a href="about.html">About Us</a>
    • Email Links: <a href="mailto:contact@gmail.com">Email Us</a>
    • Page Jumps: <a href="#section-id">Jump to Section</a> with <div id="section-id">.
    • New Tab: Use target="_blank" to open links in a new tab.

Example Code:

<a href="https://google.com" target="_blank"> Google </a> <a href="contact.html"> Contact Page </a> <a href="#footer"> Jump to Footer </a> ... <div id="footer"> Footer Content </div>
